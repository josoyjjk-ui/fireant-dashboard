⚙️ 엔진 · API 키 설정
추천 grok-4.5 (빠르고 똑똑, 비추론). 라이브 지연이 크면 grok-4.3(더 가벼움)로 교체.
Grok 단일화 완료. 받아쓰기(STT)·번역 모두 xai- 키 하나로 동작합니다. 녹음은 브라우저에서 16kHz WAV로 변환해 Grok /v1/stt로 보냅니다(Grok이 webm을 안 받아 우회). 키는 이 브라우저 localStorage에만 저장되고 api.x.ai로만 직접 전송됩니다.
